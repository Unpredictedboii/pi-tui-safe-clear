# pi-tui-safe-clear
<!-- ALL-CONTRIBUTORS-BADGE:START - Do not remove or modify this section -->
[![All Contributors](https://img.shields.io/badge/all_contributors-1-orange.svg?style=flat-square)](#contributors-)
<!-- ALL-CONTRIBUTORS-BADGE:END -->

Clears terminal scrollback/history when `pi-cli` starts and on `/quit`, **while preserving** the pi TUI header, skills list, extensions, and status bar.

## 📦 Install

```bash
pi install git:github.com/Unpredictedboii/pi-tui-safe-clear.git
```
